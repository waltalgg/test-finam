# Верстка тестового задания

Простая верстка страницы по макету Figma.

## Стек

- HTML
- CSS / SCSS
- JavaScript
- Vue.js 3

## Быстрый запуск

Установить зависимости:

```bash
npm install
```

Собрать CSS из SCSS:

```bash
npm run build:css
```

Запустить отслеживание изменений SCSS:

```bash
npm run watch:css
```

Открыть страницу:

```bash
index.html
```
